IRouteSearchPrT
Summary
This object allows for executing shortest path searches for private transport and provides access to the search results.
Object Model
Description
This object allows for executing shortest path searches for private transport and provides access to the search results. The standard method AttValue and the following ones are provided.
Examples
-
  • Python 
# This example can be used in KA.ver.
# It demonstrates how to check the network consistency between two zones,
# similar for the "Check Network" dialog.

aRouteSearchPrT = Visum.Analysis.RouteSearchPrT
aRouteSearchPrT.Clear()
aZone1 = Visum.Net.Zones.ItemByKey(1032)
aZone2 = Visum.Net.Zones.ItemByKey(125)
aNetElementContainer = Visum.CreateNetElements()
aNetElementContainer.Add(aZone1)
aNetElementContainer.Add(aZone2)
aRouteSearchPrT.Execute(aNetElementContainer, "CAR", 0)
aRouteSearchPrT.NodeChainPrT.Count # Every result >1 means: There is a path.
Public Methods
 AutozoomRoute Search Autozoom
 ClearInitializes the route search object.
 ExecuteExecutes the shortest path search for private transport. If a short path has already been found in a previous search, it will be continued. To perform a new search, first use Clear.
Public Properties

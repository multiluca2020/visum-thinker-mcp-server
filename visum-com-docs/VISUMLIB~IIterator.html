IIterator
Summary
Iterator for looping over objects of a container object
Description
All net object collections provide an IIterator object that can be used to realize loops over the objects contained in the net object container. In contrast to the property GetAll that saves all objects of a collection within an array the use of the VISUM iterator object does not depend on size limits of such arrays.
Examples
VB example: B.2.5
-
  • Python 
POICatIter=Visum.Net.POICategories.Iterator
while POICatIter.Valid:
    CurPOICat = POICatIter.Item
    # ...
    POICatIter.Next()

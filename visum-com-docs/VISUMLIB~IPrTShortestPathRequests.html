IPrTShortestPathRequests
Summary
This object contains search requests that can be passed to the Search method at IPrTShortestPathSearcher.
Create this with CreatePrTShortestPathRequests and pass the actuall requests to this object.
Public Methods
 AddRequestAdd one request.
 AddRequestsAdd multiple requests. Use this method to add multiple search requests at once without the overhead of a COM call for each request.

IFunctions
Summary
Access to general procedure settings
Object Model
Description

Corresponds to the [General procedure settings] dialog. The parameter attributes are grouped to form objects that correspond to the nodes in the tree structure. There is one access function per object. For PuT costs that occur several times ("Costs1", "Costs2", "Costs3") this access function uses the cost number as parameter. For volume settings, the access function uses th e code of the desired demand segment.

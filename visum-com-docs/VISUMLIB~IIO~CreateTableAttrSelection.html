CreateTableAttrSelection Method
Summary
Creates an ITableAttrSelection parameter object which can be used to control the layout of exported files for the methods
SaveAccessDatabaseUsingTableAttrSelection,
SaveDemandFileUsingTableAttrSelection,
SaveNetUsingTableAttrSelection,
SaveSQLiteDatabaseUsingTableAttrSelection and
SaveToMsSqlServerDatabaseUsingTableAttrSelection.
Syntax
Visual Basic
Public Function CreateTableAttrSelection() As ITableAttrSelection
Return Type
ITableAttrSelection parameter object
Description
The ITableAttrSelection can be used to control the layout of exported files like a Layout file (*.lla).

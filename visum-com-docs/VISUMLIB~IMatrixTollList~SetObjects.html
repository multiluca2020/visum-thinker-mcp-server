SetObjects Method

If true, only display active objects

Summary
This method defines for which objects of the list information is displayed
Syntax
Visual Basic
Public Sub SetObjects( _
   ByVal Value As Boolean, _
   Optional ByVal ResTrafArea As Variant _
)
Parameters
NameValue typeDescription
ValueBoolean
If true, only display active objects
ResTrafAreaVariant
Description

The method is available for most list, excluding those where no choice of displayed records is possible. The parameters of the function vary from list to list and correspond with the interactive alternatives of the list’s tool bar. In most cases, it is possible to switch between the display of all objects and the one of active objects only.

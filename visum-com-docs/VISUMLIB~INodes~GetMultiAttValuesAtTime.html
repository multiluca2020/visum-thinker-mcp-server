Description

The returned array has the structure

Array (0,0) Index 1   Array (0,1) Value 1
Array (1,0) Index 2   Array (1,1) Value 2
Array (2,0) Index 3   Array (2,1) Value 3

The indices are the indices of the objects in the list retreived by GetAll. They are returned along with the attribute values in order to identify the objects in case of only active objects are queried.
All possible attribute IDs for the individual network objects are listed in the attributes section for the network object. They are used for net object containers as well. If the attribute value is blank the return value will be blank, too.

Similar to the nomal method GetMultiAttValues, this method retreives values for a time-varying attribute valid at the request point of time for all (active) network objects at a time. For those objects which don't carry time-varyiing attribute values for the respective point of time, the value of the base attribute is returned instead.

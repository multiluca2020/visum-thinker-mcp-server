IMarking
Summary
This object provides access to the current marking state.
Description
In this context this means the current marking in the network editor. For each moment none, one or several network objects may be marked. It is possible to get the interactively marked objects as well as to mark objects via COM.
The selection state (“spatial selection”) is independent from this marking state. It can be read or set using the Active methods of network objects.

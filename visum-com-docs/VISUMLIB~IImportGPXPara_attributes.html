Names in other languages:

Short name Long name Description
DEU Faktor auf die Kosten des eingegebenen GPX-Tracks. Sind die Kosten zwischen Kandidatenstrecken zweier Eingabepunkte höher als Faktor mal Kosten plus additive Konstante, wird die Kurzwegsuche im Zielnetz abgebrochen. Bei einem Wert von 0 oder kleiner wird das ganze Netz durchsucht. Ist für die Performance relevant, falls es keinen Kurzweg zwischen zwei Strecken gibt.
ENG Factor on the cost of the entered GPX track. If the cost between candidate links of two input points is higher than factor times cost plus additive constant, the shortest path search in the target network is canceled. If the value is 0 or less, the whole network is searched. Relevant for performance if there is no shortest path between two links.

Names in other languages:

Short name Long name Description
DEU Standardabweichung Standardabweichung Gibt die Standardabweichung der (typischerweise verrauschten) Eingabedaten an. Ein großer Wert für die Standardabweichung bewirkt, dass kurze Wege bevorzugt werden. Kleine Werte der Standardabweichung bevorzugen Wege mit kleinem Abstand zu den Eingabepunkten, auch wenn der Weg selbst dadurch länger wird.
ENG StandardDeviation Standard deviation Specifies the standard deviation of the (typically noisy) input data. A large value for the standard deviation causes short paths to be preferred. Small values of the standard deviation prefer paths with a small distance to the input points, even if this makes the path itself longer.
FRA ÉcartType Écart type

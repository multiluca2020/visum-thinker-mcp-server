Return Type
Two-dimensional array containing list contents
Description

This method calculates the contents of the list specified by the chosen columns and lines (objects) and writes it to a two-dimensional array. The structure corresponds to the lines and columns of the list. Numerical contents are returned as strings. If the optional parameters fromRow and toRow are passed, the content of the list between fromRow and toRow is exported as an array. Otherwise, the entire list is exported. Both parameters are 1-based row indices. A negative fromRow has the same effect as fromRow = 1; a too large toRow has the effect of toRow = <List>.NumActiveElements (see below).

TransformationType

Numerical value internal code Summary
0 TRANSFORMATIONTYPE_NONE no transformation
1 TRANSFORMATIONTYPE_LN logarithm
2 TRANSFORMATIONTYPE_EXP exponential function
3 TRANSFORMATIONTYPE_RECIPROCAL reciprocal value
4 TRANSFORMATIONTYPE_TRANSPOSE transpose the matrix
5 TRANSFORMATIONTYPE_MIRROR_LOWER mirror the lower triangle
6 TRANSFORMATIONTYPE_MIRROR_UPPER mirror upper triangle
7 TRANSFORMATIONTYPE_SYMMETRIZE symmetrize matrix

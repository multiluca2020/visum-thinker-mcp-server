Check Method
Perform "forced" block check
Also consider minor errors as failure
Summary
Performs block check for block.
Syntax
Visual Basic
Public Function Check( _
   Optional ByVal Forced As Boolean = False, _
   Optional ByVal ConsiderMinorFaults As Boolean = False _
) As Boolean
Parameters
NameValue typeDescription
ForcedBooleanPerform "forced" block check
ConsiderMinorFaultsBooleanAlso consider minor errors as failure
Return Type
Only minor or no errors found
Description
Optionally, you can trigger a "forced" block check. The value returned is TRUE, if only minor errors (or no errors, if considerMinorFaults is true) were found.

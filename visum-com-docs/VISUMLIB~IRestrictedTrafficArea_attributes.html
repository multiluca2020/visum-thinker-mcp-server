Short name Long name Description
DEU GesperrtesVSysSet Gesperrtes VSys-Set In der Umlegung werden alle Strecken, die zum regulierten Verkehrsbereich gehören, für die Verkehrssysteme aus der angegebenen Menge gesperrt. Dies gilt entweder für jeden Verkehr (für Verkehrsbereiche vom Typ Fahrverbot) oder nur für Durchfahrtsverkehr (für Verkehrsbereiche vom Typ Durchfahrtsverbot). Für andere Typen von Verkehrsbereichen ist das Attribut nicht relevant.
ENG BlockedTSysSet Blocked TSys set During the assignment, all links belonging to the restricted traffic area are blocked for the specified set of transport systems. This applies either to all traffic (for traffic areas of the type “no traffic”) or only to through traffic (for traffic areas of the type “no through traffic”). The attribute is not relevant for other types of traffic areas.
FRA EnsSysTrInterdit Ensemble SysTr interdit

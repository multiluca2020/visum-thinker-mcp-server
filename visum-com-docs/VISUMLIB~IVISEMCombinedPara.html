IVISEMCombinedPara
Summary
Access to base parameters of the VISEM combined trip distribution / mode choice operation.
Object Model
Description
The trip distribution and mode choice parameters can be found in subordinate objects
Examples
-
  • Python 
# Matching the example
# c:\Users\Public\Documents\PTV Vision\PTV Visum 2024\Examples\Demand Visem\Visem_Result.ver
# This example provides access to the Utility function of Person group EC and Mode C:

myOperation = Visum.Procedures.Operations.ItemByKey(9)
myOperation.VISEMCombinedParameters.VISEMModeChoiceParameters.VISEMDGroupModeChoiceParameters("EC").VISEMDGroupModeModeChoiceParameters("C").AttValue("UtilityFunction")

myUtilityFunction = 'CONTEXT[MODE\\TT_COEF] * 0.95 *  Matrix([CODE] = "TTC" & [MATRIXTYPE] = 4 & [DSEGCODE] = "C") + CONTEXT[MODE\\WALK_COEF] * (FROM[ACCESSTIMECAR] + TO[EGRESSTIMECAR]) + CONTEXT[PERSONGROUP\\C_SPECIFIC_CONSTANT]'

myOperation.VISEMCombinedParameters.VISEMModeChoiceParameters.VISEMDGroupModeChoiceParameters("EC").VISEMDGroupModeModeChoiceParameters("C").SetAttValue("UtilityFunction", myUtilityFunction)

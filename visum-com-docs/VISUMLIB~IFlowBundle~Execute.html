Execute Method
Network object container containing the network objects used for flow bundle calculation
Summary
Executes a flow bundle calculation.
Syntax
Visual Basic
Public Sub Execute( _
   ByVal NetElms As INetElements _
)
Parameters
NameValue typeDescription
NetElmsINetElementsNetwork object container containing the network objects used for flow bundle calculation
Description
The net elements to be evaluated are contained in a net element container of type INetElements. Each (admissible) object in this container defines a condition. These can be linked by "OR" operations. Calling this method closes the term for new conditions, i.e. conditions added previously or later are linked by an "OR" operation. This allows you to link several "AND" linked terms by an "OR" operation. To do so, call the Execute method several times in a row, without using a Clear.
If you wish to limit the activity, for each network object, call the CreateCondition method with the corresponding IActivityTypeSet object.

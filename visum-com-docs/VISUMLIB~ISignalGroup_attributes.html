Short name Long name Description
DEU GrünEnde Grünzeitende Das Grünzeitende der Signalgruppe liegt im Intervall [0, Umlaufzeit]. Der Wert ergibt sich bei einer phasenbasierten LSA aus den Grünzeiten der Phasen. Ist bei einer signalgruppenbasierten LSA für Grünzeitstart und Grünzeitende der Wert 0 angegeben, zeigt die Signalgruppe Dauerrot, bei jedem anderen, aber gleichen Wert Dauergrün. Relevant für die ICA-Berechnung ab HCM2000.
ENG GtEnd Green time end The green time end of the signal group lies in the interval [0, cycle time]. In the case of a stage-based signal controller, the value results from the green times of the stages. If the value 0 is specified for the green time start and green time end for a signal group-based signal controller, the signal group shows permanent red, and permanent green for any other but the same value.
FRA FinTVert Fin temps de vert

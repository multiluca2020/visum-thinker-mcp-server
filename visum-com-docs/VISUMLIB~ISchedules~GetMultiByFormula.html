GetMultiByFormula Method
Expression of the formula to be evaluated as a string. Must evaluate to a numerical value.

Summary
Evaluates a formula for all (active) objects of the collection.
Syntax
Visual Basic
Public Function GetMultiByFormula( _
   ByVal Formula As String, _
   Optional ByVal OnlyActive As Boolean = True _
) As Variant
Parameters
NameValue typeDescription
FormulaStringExpression of the formula to be evaluated as a string. Must evaluate to a numerical value.
OnlyActiveBooleanControls whether to return formula results only for active network objects
Return Type

The results are returned as a 2-dimensional array of the form [ [ID, Value] ].

Description

Evaluates a formula expression for all active objects of the collection. Attribute values remain unchanged. The formula must evaluate to a numeric value, string expressions are not yet supported.
 

Examples
-
  • Python 
linklength = Visum.Net.Links.GetMultiByFormula(formula = '[LENGTH] * 1000', OnlyActive=True) #returns length in meter for every active link

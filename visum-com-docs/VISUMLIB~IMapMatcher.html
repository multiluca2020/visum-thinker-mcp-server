IMapMatcher
Summary
IMapMatcher provides methods and objects to allow the mapping of external coordinates on link sequences in the current Visum net.
Object Model
Description

Use the CreateMapMatcher Method to create a MapMatcher instance for the currently loaded network. Multiple MapMatchers (with different parameters) can be created and used for the same network.

Please Note: The 'Incremental' matching algorithm has been removed in PTV Visum 18. Only the 'Global' algorithm is available and the attribute 'Algorithm' for selecting the matching algorith is therefore obsolete.

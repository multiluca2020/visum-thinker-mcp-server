IAttribute
Summary
An IAttribute-object contains the definition of a predefined or user-defined attribute of a net object or a list.
Object Model
Description
It gives access to properties of the attribute, which can be used in a program e.g. doing operations on attributes with certain properties, without naming the attributes. For all properties except ValueType, Editable and Category methods for setting the value are provided. These make sense for user defined attributes only: Since these properties are fixed for predefined attributes an error message will be generated when trying to change these properties for predefined attributes.

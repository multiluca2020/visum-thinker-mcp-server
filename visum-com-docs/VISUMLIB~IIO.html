IIO
Summary
Access to open/save/input/output interfaces.
Object Model
Description

This class serves as container for many open, save, import and export related methods, corresponding to the File menu entries
- Open
- Open file
- Save as
- Save file as
- Import
- Export
For a respective detailed description refer to the Online Help.

 

Note that the current naming convention does not cover all methods, some are already in service since many years.
The first set of methods following the naming conventions "Create" or "Create..Para" serves to mirror the respective dialog, in which open/import/export parameters can be set.

The next set of "Export..." and "Import..." methods are corresponding to the import and export interfaces, but note that ImportPuTProject combines multiple interfaces.

The "Load..." methods correspond partly to "Open"/"Open file" and partly to "Import".

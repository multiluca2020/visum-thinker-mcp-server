RemoveAll Method
Delete only active objects
Summary
Removes all (active) objects that are contained in the container
Syntax
Visual Basic
Public Sub RemoveAll( _
   Optional ByVal OnlyActive As Boolean = True _
)
Parameters
NameValue typeDescription
OnlyActiveBooleanDelete only active objects
Description
Note that removing objects is not possible for some object types (e.g. turns, directions). If the method is called on a container of objects of one of these types, an error message is generated. The call is allowed for all object types for which a (single) “remove” method can be found at the INet obj

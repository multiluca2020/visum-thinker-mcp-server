IOperationExecutor
Summary
The interface IoperationExecutor offers the same functionality as the procedure flow view.
Object Model
Description

Analog to the GUI the following options are possible:

  • Mark current procedure step
  • Single-step processing: Execute the current procedure
  • Reset the procedure state
  • Ask current procedure
  • Ask if procedure flow is finished

IActivityTypeSet
Summary
This object allows you to set the kind of traffic when calculating flow bundles
Description
 IActivityTypeSet is used to control the kind of traffic (inbound, outbound, through,...) to be considered when computing flow bundle,.equivalent to the tab "Traffic types" (PrT) and "Passenger types" (PuT) in the dialog "Edit flow bundle item".
Examples
-
  • Python 
myActivityTypeSet = myFlowBundle.CreateActivityTypeSet()
myActivityTypeSet.Add(124) # all kinds of PuT Passenger types: 4 + 8 + 16 + 32 + 64 = 124
Public Methods

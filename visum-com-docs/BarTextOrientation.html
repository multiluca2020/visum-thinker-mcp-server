BarTextOrientation

Bar text orientation as controlled by attribute BarTextOrientation of IBarTextFormatGPA Object

Numerical value internal code Summary
0 BarTextOrientation_Horizontal Bar text orientation horizontal
1 BarTextOrientation_Vertical Bar text orientation vertical
2 BarTextOrientation_VerticalWithSum Bar text orientation vertical with sum
3 BarTextOrientation_OnlySum Bar text orientation only sum
4 BarTextOrientation_Matrix Bar text orientation matrix

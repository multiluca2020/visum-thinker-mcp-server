INetReadRouteSearch
Summary
INetReadRouteSearch Interface
Description

INetReadRouteSearch sets the options of all transport systems or of a specific transport system for completing the line and system route paths when reading the net additively.

It is created through the CreateNetReadRouteSearch Method

Public Methods
 LoadOpens file with parameter settings
 SetForAllTSysSets the same parameters for all transport systems
 SetForTSysSets the parameters for one single transport system

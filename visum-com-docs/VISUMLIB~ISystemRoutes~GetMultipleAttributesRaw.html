Return Type
Two-dimensional SafeArray containing selected attribute values for all network objects
Description

The attribute ID have to be specified as a vector of strings. The return value contains a matrix of VARIANT values consisting of the values of all objects and specified attributes. The rows of the matrix correspond to the net objects in standard order (ordered by their keys). The columns correspond to the attributes in the order as specified.

This method allows high performance access to numeric attribute values. It is best used in conjunction with Python/numpy using the VisumPy.helpers.GetMultipleAttributesRaw function.

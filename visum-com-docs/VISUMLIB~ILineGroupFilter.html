ILineGroupFilter
Summary
The ILineGroupFilter object is a hierarchical line filter, equivalent to the dialog "Filter for PuT lines".
Object Model
Description

You can set the filter for the individual filter levels of your choice. The filter operations are managed using one IGroupElementFilter object per hierarchy level. The ILineGroupFilter object itself does not provide any basic functions.

The UseXXX-Properties specify whether to apply the filter to that level of the line hierarchy and the methods are used to access specific filter objects

Examples
-
This example creates first a condition to filter Vehicle journey items for a certain Stop, querying the Line route item behind it. This spatial condition is combined over an "AND" operator with two temporal conditions limiting the arrival and departure on a certain time span.
  • Python 
Visum.Filters.InitAll()
aLineGroupFilter = Visum.Filters.LineGroupFilter()
aVehJourneyItemFilter = aLineGroupFilter.VehJourneyItemFilter()
aStopNo = 1

aVehJourneyItemFilter.AddCondition(r'OP_NONE', False, \
r'TIMEPROFILEITEM\LINEROUTEITEM\STOPPOINT\STOPAREA\STOPNO', \
'EQUALVAL', aStopNo)

aFromTime = 6 * 60 * 60
aToTime = 8 * 60 * 60

aVehJourneyItemFilter.AddCondition(r'OP_AND', False, r'ARR', \
r'ContainedIn', Visum.Filters.Range(aFromTime, aToTime))

aVehJourneyItemFilter.AddCondition(r'OP_OR', False, r'DEP', \
r'ContainedIn', Visum.Filters.Range(aFromTime, aToTime))

aVehJourneyItemFilter.AddBrackets(2, 3)

aLineGroupFilter.UseFilterForVehJourneyItems = True

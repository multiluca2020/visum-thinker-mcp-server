Description

Inserts a time profile of given name into the specified line route. The following default values are used:

  • On all route points of the line route time profile items are created.
  • At all time profile items that correspond to a stop point, board and alight are allowed (at the first item board only, on the last alight only).
  • The PostRunTime of a time profile item is set to the link travel time of the transport system line for the traversed section. At the last item it is zero.
  • The stop time of an item corresponding to a stop point is set to the standard stop time of this stop point.

All these values can be changed subsequently. In particular, time profile items can be added or removed once the time profile exists.

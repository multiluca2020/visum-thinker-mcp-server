Return Type
The generated system route
Description
Inserts a system route with the specified name and transport system into the network. The nodes and stop points are given in the net element container of type INetElements. Between these points a shortest path search is executed. The search parameters are determined by an INetReadRouteSearchTSys object analogous to additive net reading.

PasteFromClipboard Method
Summary
This method allows to paste content from the clipboard via COM in lists.                            
Syntax
Visual Basic
Public Sub PasteFromClipboard()
Examples

-
  • Python 
LinkList = Visum.Lists.CreateLinkList
LinkList.Show()
LinkList.AddColumn("LENGTH")
LinkList.AddColumn("NAME")
# fill clipboard
LinkList.SaveToClipboard(59)

# set content from clipboard to list
LinkList.PasteFromClipboard()

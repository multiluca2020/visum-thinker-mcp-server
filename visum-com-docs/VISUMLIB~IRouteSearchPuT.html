IRouteSearchPuT
Summary
This object allows for executing table based shortest path searches for public transport.
Object Model
Description

This object allows for executing timetable based shortest path searches for public transport.

The result can be analyzed through the PuT path search list, see Visum.Lists.CreatePuTPathSearchLegList 

Public Methods
 AutozoomThe network view is zoomed so that the path found is displayed optimally.
 ClearInitializes the route search object.
 CreateRouteSearchPuTParametersCreate parameter object for timetable-based PuT route search.
 ExecuteExecutes a table based shortest path search for public transport.

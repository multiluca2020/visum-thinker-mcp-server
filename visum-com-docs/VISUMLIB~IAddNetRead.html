IAddNetRead
Summary
Object controling additional net reading
Description
When an additional network is read interactively certain options for conflict handling can be set for each table in the dialog. These options can be accessed and controled via COM with the object AddNetRead after it was created with Visum.CreateAddNetReadController. Options can be set either for all tables or for a single table. A single table is identified by its ID which is listed in the document ATTRIBUTE.XLS (in the Visum document list). The points of interest form an exception. All tables with an ID POIOFCAT_... are set with the table ID „POI“.
The get and set methods NumericOffset, UseNumericOffset, AlphaPrefix and UseAlphaPrefix are properties, i.e. they can be used in VBA with the “myAddNetRead.UseNumericOffset(“STOPPOINT“) = True” syntax.
Examples
VB example: B.1.3

Public Properties

 OfferAlphaPrefixIndicates whether alphanumerical conflict aviodance can by applied for a table
 OfferNumericOffsetIndicates whether numerical conflict aviodance can by applied for a table
 UseAlphaPrefixReturns a value which indicates whether a table can use an alphanumeric prefix or not
 UseNumericOffsetReturns a boolean value which is true if the use of a numeric offset is currently set for the specified table.

Names in other languages:

Short name Long name Description
DEU BeschrPosRelX Beschriftungsposition relativ (x) [mm] Verschiebung der Anzeigeposition der Tabelle in x-Richtung ausgehend von der Position des GIS-Objekts. Die Verschiebung wird in absoluter Größe (Einheit [mm]) angegeben, hängt also nicht von der Zoomstufe ab.
ENG LabelPosRelX Label position relative (x) [mm] Shift the display position of the table in direction x starting from the position of the GIS object. The shift is specified in absolute size (unit [mm]), so it does not depend on the zoom level.
FRA PosRelLibelléX Position du libellé relative (x) [mm] Décalage de la position d’affichage de la table dans la direction x à partir de la position de l’objet SIG. Le décalage est spécifié en taille absolue (unité [mm]), il ne dépend donc pas du niveau de zoom.

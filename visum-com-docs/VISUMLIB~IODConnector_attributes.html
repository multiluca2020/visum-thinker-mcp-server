DEU UmlgAnzKonvergierterIterationen Anzahl der aufeinanderfolgenden Iterationen, in der das Netzobjekt die Konvergenzbedingung erfüllte Anzahl der aufeinanderfolgenden Iterationen, in der das Netzobjekt die Konvergenzbedingung sowohl für die Belastung als auch für den Widerstand erfüllte. Das Attribut wird von allen statischen IV-Gleichgewichtsumlegungen (Gleichgewichtsumlegung, LUCE, Bi-conjugate Frank-Wolfe, Lernverfahren, TRIBUT-Lernverfahren) sowie der Umlegung mit ICA bei eingestellten WebTAG-konformen Konvergenzkriterien verwendet.
ENG AssignNumConvergedIterations Number of successive iterations in which the network object fulfilled the convergence condition Number of consecutive iterations in which the network object satisfied the convergence condition for both the volume and the impedance. The attribute is used by all static PrT equilibrium assignments (Equilibrium assignment, LUCE, Bi-conjugate Frank-Wolfe, Equilibrium_Lohse, TributEquilibrium_Lohse) as well as the assignment with ICA when WebTAG-compliant convergence criteria are set.
FRA AffectationNbItérationsConvergence Nb itérations successives dans lesquelles l’élément de réseau a satisfait au critère de convergence Nombre d’itérations successives dans lesquelles l’élément de réseau a satisfait au critère de convergence aussi bien pour la charge que pour le temps généralisé. L’attribut est utilisé par toutes les affectations à l’équilibre TI (affectation à l’équilibre, LUCE, Bi-conjugate Frank-Wolfe, méthode d’apprentissage, méthode d’apprentissage TRIBUT) ainsi que par l’affectation avec ACI avec critères de convergence conformes à WebTAG configurés.

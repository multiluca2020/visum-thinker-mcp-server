Compare operators

Comparison operators used in filter expressions

Numerical value internal code Summary
0 None No comparison
1 LessVal Less than given value
2 LessAtt Less than given attribute
3 GreaterVal Greater than given value
4 GreaterAtt Greater than given attribute
5 LessEqualVal Less than or equal to given value
6 LessEqualAtt Less than or equal to given attribute
7 GreaterEqualVal Greater than or equal to given value
8 GreaterEqualAtt Greater than or equal to given attribute
9 EqualVal Equal to given value
10 EqualAtt Equal to given attribute
11 NotEqualVal Not equal to given value
12 NotEqualAtt Not equal to given attribute
13 ContainedIn Contained in given set
14 ContainsOneOf Contains one of given set
15 ContainsAll Contains all of given set
16 ContainsNone Contains none of given set

IAddInParameter
Summary
Represents the parameters for an AddIn
Description

This object is used to exchange parameter values between Visum and the dialog part and the body code of the AddIn. A singleton object of this type is accessible through the global variable 'Parameter' when the dialog or body code of an AddIn is executed.

For AddIns which are executed in the procedure sequence, it can also be accessed through the attribute AddInParameter of the IAddInPara Object. 

Public Properties
 DataActual parameters values stored as a single string. The contents and coding of this string is specific to the respective AddIn. This is usually encoded/decoded in formats like cPickle or JSON.
 OKIndicates whether the parameters are valid. When the dialog code of an AddIn sets this to false, changed values will not be stored in Visum.
 OperationNoOperation number of the AddIn operation calling the AddIn. This is only used when AddIn are called in the procedure sequence.

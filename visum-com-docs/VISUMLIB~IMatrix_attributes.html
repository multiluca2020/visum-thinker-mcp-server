Names in other languages:

Short name Long name Description
DEU MatrixTyp Matrix-Typ Typ der Matrix: Eine Nachfragematrix enthält Nachfrage in Anzahl Personen, Anzahl Fahrzeuge oder Anzahl Aufträge. Eine Kenngrößenmatrix enthält die Werte einer Kenngröße und hat verschiedene Einheiten. Eine externe Matrix enthält extern zugespielte Daten.
ENG MatrixType Matrix type Type of matrix: A demand matrix contains demand in number of persons, number of vehicles, or number of orders. A skim matrix contains values of a skim and has different units. An external matrix contains externally fed data.
FRA TypeMatrice Type de matrice Type de matrice : une matrice de la demande contient la demande en nombre d’usagers, de véhicules ou d’ordres. Une matrice d’indicateurs contient les valeurs d’un indicateur et comporte différentes unités. Une matrice externe contient des données enregistrées en externe.

>> Visum - COM > ICalendarPeriod
ICalendarPeriod
Summary
AddLinThis object provides access to the calendar settings.
Object Model
Description
There is no collection object since there is only one calendar in each network.
Examples
-
  • Python 
timeInterval = Visum.Net.CalendarPeriod.TimeInterval('0900') #or .TimeInterval(1) by index or code
timeInterval.AttValue("NAME") #gives '09:00:00-09:30:00'
currtimeInterval = Visum.Net.CalendarPeriod.CurrentTimeInterval
currtimeInterval.AttValue("NAME") #gives '08:30:00-09:00:00'

#set current time interval to chosen time interval
Visum.Net.CalendarPeriod.CurrentTimeInterval = timeInterval
currtimeInterval.AttValue("NAME") #gives '09:00:00-09:30:00'

# get a sub attribute of a net object for the "current time interval"
Visum.Net.Links.ItemByKey(FromNodeNo, ToNodeNo).AttValue("VOLVEHPRT(&CI)")

IGravitationPara
Summary
This object provides access to the parameters of the distribution model within the trip distribution procedure
Examples
-
  • Python 
proc = Visum.Procedures.Operations.AddOperation(1)
proc.SetAttValue("OPERATIONTYPE",46)  # Trip distribution
paras = proc.TripDistributionParameters.TripDistributionDStratParameters("CRA_car") 
paras.GravitationParameters.SetAttValue("PARA_C",-1)

Return Type
Two-dimensional SafeArray with the object indices and attribute values
Description

The returned array has the structure

Array (0,0) Index 1 Array (0,1) Value 1
Array (1,0) Index 2 Array (1,1) Value 2
Array (2,0) Index 3 Array (2,1) Value 3

The indices are the indices of the objects in the list retreiveved by GetAll. They are returned along with the attribute values in order to identify the objects in case of only active objects are queried.
All possible attribute IDs for the individual network objects are listed in the attributes section for the networ object. They are used for net object containers as well. If the attribute value is blank the return value will be blank, too.

Using this function, any attribute of a net object can be obtained. All attribute IDs are listed in the attribute section. Sub-attributes are added in parentheses, separated by comma if more than one. If the attribute value is blank the return value will be blank, too.
(e.g. "VOLPERSPUT(OV,AP)").

IExportShapeFilePara
Summary
Use this object to set parameters to control the shapefile export.
Description
It is needed as a parameter of the IVisum.ExportShapefile method and is created using the method CreateExportShapeFilePara. You can use the standard method AttValue to set individual parameter values.
Public Methods
 AddColumnAdd a column to the current layout and defines its format
 AddKeyColumnsThis method adds one or more columns corresponding to the key values of the objects to the end of layout.
 AddPOICategoryRestrictionAdds a POI category to the restrictions for the export
 ClearLayoutClear the current export (list) layout
 ClearPOICategoryRestrictionClears the restrictions of the export to any POI categories
 OpenLayoutThis method reads a list layout from file to be used for shapefile export.
Public Properties
 AttValueGets or sets an attribute value of the network object
 ObjectTypeGets or sets the object type to be exported

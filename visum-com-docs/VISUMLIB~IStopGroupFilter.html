IStopGroupFilter
Summary
The IStopGroupFilter object is a hierarchical stop filter.
Object Model
Description

 You can set the filter for the individual filter levels of your choice. The filter operations are managed using one IGroupElementFilter object per hierarchy level. The IStopGroupFilter object itself does not provide any basic functions.

The UseXXX-Properties specify whether to apply the filter to that level of the stop hierarchy and the methods are used to access specific filter objects

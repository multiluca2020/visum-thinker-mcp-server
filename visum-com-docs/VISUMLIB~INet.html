INet
Summary
Represents the network loaded in Visum. Contains all net objects of the Visum network.
Object Model
Description

The INet object represents a Visum network. Methods involving the network as a whole are provided by the INet object. Below INet objects are situated representing elements of the network.

Besides some other methods and properties, the INet object provides properties for accessing all network object collections and methods for inserting and deleting net objects. These methods are all formed in the same manner. The remove method takes the net object to be removed as input parameter. The name of the method is composed by „Remove“ and the name of the net object, e.g. „RemoveNode ( [in] INode*)”. The add method is used to add new net elements to the network. It takes as input parameters the values of all attributes that are necessary for insert and returns the new net object as a result. A valid key number is not necessary. Visum generates the key automatically in case of a zero or negative numeric value. Optionally coordinates can be specified for certain network objects. For the net objects INotepadLines, ILinkType, ITurn, ITerritoryPuTDetail and IDirection there are no add or remove methods.

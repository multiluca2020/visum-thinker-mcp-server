ITableAttrSelection
Summary
Defines the layout (tables and columns) to be used in export functions
Description

Defines the layout (tables and columns) to be used in variuos export functions such as SaveNetUsingTableAttrSelection, SaveAccessDatabaseUsingTableAttrSelection etc.

Multiple layout objects can be created using the CreateTableAttrSelection Method and used as needed. The tables are identified by their names (IDs) as used in the (english) *.net file. Tables are added to the layout implicitely when any of the methods for adding columns is called with the respective table ID.

Public Methods
 AddAllColumnsAdds all columns of a table to the layout
 AddAllPOICategoriesAdds tables for all POI categories
 AddColumnAdds a single column to the layout
 AddColumnsForAllUserDefinedAttributesAdds columns for all user-defined attributes of a table
 AddColumnsForCategoryAdds all columns of a specific category
 AddColumnsForUserDefinedGroupAdd columns for user defined group (experimental)
 AddCommonUseCaseAreasIncludes all tables and columns for the typical use case 'Areas'
 AddCommonUseCaseBlocksIncludes all tables and columns for the typical use case 'Line blocks'
 AddCommonUseCaseGraphicAndPresentationIncludes all tables and columns for the typical use case 'Graphics and presentation'
 AddCommonUseCaseMainNodesIncludes all tables and columns for the typical use case Main nodes
 AddCommonUseCaseTimetableIncludes all tables and columns for the typical use case 'Timetable'
 AddDefaultColumnsAdds all default columns of all included tables or of a specific table
 AddKeyColumnsAdds the key columns of a specific table
 ClearResets the entire layout
 OpenLayoutFilePopulates the layout object from a layout file
 RemoveColumnRemoves a column from the layout
 RemoveNonEditableAttributesRemoves all non-editable attributes from all tables contained in the layout
 RemoveTableRemoves a table from the layout
 SaveLayoutFileSaves the layout to a file
Public Properties
 OmitDefaultColumnsGet value for OmitNonDefaultColumns
 OmitEmptyTablesControls whether tables with only empty columns - at the time of the export - shall be included

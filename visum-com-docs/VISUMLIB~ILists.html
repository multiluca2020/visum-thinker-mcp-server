ILists
Summary
ILists is (analogous to INet) an intermediate object that exclusively serves for the creation of specific exemplars of lists. All further operations during the creation of the lists are executed on the specific list object.
Object Model
Description

IList contains a method for any existing list in Visum which creates an empty entity of the specific list. The lists contents and visualization are created using methods of the specific list object.

To access the contents of a list in Visum via the COM interface, the following steps have to be handled in any case:

  1. create new entity of a list,
  2. define the set of objects,
  3. choose columns and define their properties,
  4. export the contents to one of the target media

The access a single list always uses the ILists object. It is reached from the IVisum object through the Lists Property. Individual lists are created from the ILists object. Since the interface of all individual lists is widely similar, the introduction is split into the common list interface and special methods of some single lists.

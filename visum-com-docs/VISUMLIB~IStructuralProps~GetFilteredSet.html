GetFilteredSet Method
Formula defining the logical expression for the filter
Summary
Retreives a fixed collection of objects from this container fulfilling a filter condition at the time of the query
Syntax
Visual Basic
Public Function GetFilteredSet( _
   ByVal Formula As String _
) As IStructuralProps
Parameters
NameValue typeDescription
FormulaStringFormula defining the logical expression for the filter
Return Type

Returns a new container of the same type containing only those objects for which the logical expression provided as the condition returns True or a value > 1. The expression is defined in usual formula syntax, making use of all direct and indirect attributes of the objects.

As opposed to the FilteredBy,  the filter expression is evaluated only once at the time of the query. Changes to attribute values etc. done after calling GetFilteredSet don't change the returned collection.

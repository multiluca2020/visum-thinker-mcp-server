IIsochrones
Summary
Executes the calculation of isochrones for PrT or PuT.
Object Model
Description
This class serves to preparare and execute the calculation of isochrones for PrT or PuT. It corresponds to the Dialog "Graphics tools (Isochrones)".
Examples
-
  • Python 
nodeList = Visum.Lists.CreateNodeList
basePara = Visum.Analysis.Isochrones.CreatePuTIsochroneBaseParameters()
basePara.SetMode("TC")
basePara.SetAttValue("MaxNumTransfers", 0)

requestPara = Visum.Analysis.Isochrones.CreatePuTIsochroneRequestParameters()
requestPara.SetAttValue("SearchIntervalStartTime", 7*3600)
requestPara.SetAttValue("SearchIntervalEndTime", 8*3600)
requestPara.SetAttValue("ExtensionTime", 1*3600)

requestPara.SetAttValue("SearchDirection", 0)
Visum.Analysis.Isochrones.Clear()
Visum.Analysis.Isochrones.ExecutePuTWithParameterObjects(netelems, basePara, requestPara)
nodeList.SaveToAttributeFile("IsochronesWithParameterObjectsForward.att", 59)

requestPara.SetAttValue("SearchDirection", 1)
Visum.Analysis.Isochrones.Clear()
Visum.Analysis.Isochrones.ExecutePuTWithParameterObjects(netelems, basePara, requestPara)
nodeList.SaveToAttributeFile("IsochronesWithParameterObjectsBackward.att", 59)

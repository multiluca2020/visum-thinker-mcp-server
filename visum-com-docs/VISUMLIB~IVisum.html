IVisum
Summary
Top-level object of the Visum-COM-interface
Object Model
Description

IVisum is the top-level object; all other objects are linked to IVisum .

Create an IVisum object

Use the CreateObject(ObjectName) command in VBA or similar commands in other programming languages .

Delete an IVisum object

In Visual Basic, an object is deleted by setting it to "nothing". If all created COM-objects are deleted in this manner, the VISUM instance started via COM will close itself automatically.

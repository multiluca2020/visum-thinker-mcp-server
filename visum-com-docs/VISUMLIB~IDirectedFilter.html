IDirectedFilter
Summary
An IDirectedFilter is a simple filter with a special function: it filters in one direction only
Object Model
Description
Accordingly, this allows you to switch between directed and undirected filtering. It provides basic filter functions as well as the following properties.
Public Methods
 AddBracketsWraps a set of conditions in brackets
 AddConditionAdds an additional condition to the filter.
 AddFormulaConditionAdd Condition
 AddFormulaRelationConditionAdd Condition
 AddRelationConditionAdds a relation condition to the bracketed condition
 BracketsReturns the brackets defined in this filter
 ConditionReturns a specific condition
 GetCombinationOpWithPrevConditionGet the combination operation with the predecessor condition
 InitInitializes the filter, i.e. deletes all filter conditions.
 NumConditionsReturns the number of conditions in this filter
 RemoveBracketsRemoves a pair of brackets
 RemoveConditionRemoves the specified condition from the filter
 RemoveConditionsRemoves all conditions from the filter
 SetCombinationOpWithPrevConditionSet the combination operation with the predecessor condition
Public Properties
 ComplementSwitches the entire filter effect (not the individual filter operation) from direct to complementary
 UndirectedSwitches between a directed (false) and an undirected (true) filter
 UseFilterSwitches the filter on or off

ISingleFilter
Summary
The ISingleFilter is a single-level filter as used e.g. for nodes or zones
Object Model
Description
 It provides basic filter functions as well as the following properties
Examples
-
  • Python 
myNodeFilter = Visum.Filters.NodeFilter()
myNodeFilter.Init()
myFirstCondition = myNodeFilter.AddCondition(Op="OP_NONE", Complement=False, AttID="TypeNo", CompareOperator=13, Val=1, Position=0)
mySecondCondition = myNodeFilter.AddCondition(Op="OP_AND", Complement=False, AttID="ControlType", CompareOperator=13, Val=0, Position=1)
myNodeFilter.GetCombinationOpWithPrevCondition(1)
'OP_NONE'
myNodeFilter.GetCombinationOpWithPrevCondition(2)
'OP_AND'

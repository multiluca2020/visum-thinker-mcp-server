Condition combination

Logical operators used to combine multiple conditions in a filter. Use the identifier as a string to select the operator

Identifier Operation
OP_NONE This operation must be specified with the first filter condition. If a condition has already been set, an error message appears.
OP_OR OR operation with the previous condition.
OP_AND AND operation with the previous condition.

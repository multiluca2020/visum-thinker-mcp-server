IPrTShortestPathSearcher
Summary

This object allows for executing shortest path searches for private transport and provides access to the search results. Unlike IRouteSearchPrT, which is the COM interface to the shortest path search in the Visum graphics menu, this interface is designed for executing many shortest path queries very fast.

Object Model
Examples
-
  • Python 
# create searcher
searcher = Visum.Net.CreatePrTShortestPathSearcher()

# create requests
requestList = []
for ...:
    # cost attribute can be either an enum PrTSearchCriterionT (t0 = 0, tCur = 1, impedance = 2 ...)
    # or the string of a attribute ID for links, connectors, turns, main turns
    fromNetElem = Visum.Net.Nodes.ItemByKey(...)
    toNetElem = Visum.Net.Nodes.ItemByKey(...)
    tSys = "C"
    costAttribute = 2 # 2 == impedance 
    requestList.append([fromNetElem, toNetElem, tSys, costAttribute])

requests = searcher.CreatePrTShortestPathRequests()
requests.AddRequests(requestList)

# run search
pathAttributes = [1, "AddVal1"] # 1 == tCur
searchResults = searcher.Search(requests, pathAttributes)

# searchResults.Costs: path costs
# searchResults.AdditionalAttributeValues: values for path attributes

IEVAWeightingPara
Summary
Access to parameters of the EVA weighting operation
Object Model
Examples
-
Setting the parameters of the EVA weighting procedure. The example is based on the installation example "%public%\Documents\PTV Vision\PTV Visum 13\Examples\Demand_EVA\ver_EVA.ver"
  • Python 
evaWeightingOp = Visum.Procedures.Operations.ItemByKey(10) #retreive the operation

#here the subordinate parameters are accessed hierarchically
#but, this could also be written in one line as: evaDStratModeAssessmentPara = evaWeightingOp.EVAWeightingParameters.GetModeParameters("O").GetModeAssessmentParameters("JOURNEYTIME").GetModeAssessmentDStratParameters("M101_WA")
evaModePara = evaWeightingOp.EVAWeightingParameters.GetModeParameters("O")						 #'O' is the code of the mode
evaModeAssessmentPara = evaModePara.GetModeAssessmentParameters("JOURNEYTIME")		        	 #'JOURNEYTIME' is the built-in assessment type Journey Time
evaDStratModeAssessmentPara = evaModeAssessmentPara.GetModeAssessmentDStratParameters("M101_WA") #'M101_WA' is the code of the demand stratum
evaDStratModeAssessmentPara.SetAttValue("Para_a", 100) #set 'Para_a' to 100

#set parameter for a custom assessment type
evaModeAssessmentPara2 = evaModePara.GetModeAssessmentParameters("Anpassungszeit")		        	 #'Anpassungszeit' is a custom the assessment type
evaDStratModeAssessmentPara2 = evaModeAssessmentPara2.GetModeAssessmentDStratParameters("M101_WA") #'M101_WA' is the code of the demand stratum
evaDStratModeAssessmentPara2.SetAttValue("Para_a", 100) #set 'Para_a' to 100

IEVAModeChoicePara
Summary
Access to parameters of the EVA distribution/mode choice operation
Object Model
Description
Actual parameters are distributed across several subordinate objects reflecting the structure of the dialog. They are accessed through an intermediate EVADStratModeChoiceParameters object per demand stratum.

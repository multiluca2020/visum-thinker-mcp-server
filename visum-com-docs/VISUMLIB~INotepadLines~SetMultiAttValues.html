Description

The input parameter newValues specifies the Values to be handled with an array of the structure

Array (0,0) Index 1   Array (0,1) Value 1
Array (1,0) Index 2   Array (1,1) Value 2
Array (2,0) Index 3   Array (2,1) Value 3

which is equivalent to the structure returned by GetMultiAttValues. The set of network objects to be modified is thereby not controlled through the 'Active'-state, but through the indices of the network objects used in the newValue parameter.

The indices are the indices of the objects in the list as retreived by GetAll. The array can be generated with the function GetMultiAttValues
All possible attribute IDs for the individual network objects are listed in the attributes section for the network object. They are used for net object containers as well.

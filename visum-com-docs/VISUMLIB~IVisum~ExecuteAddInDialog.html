ExecuteAddInDialog Method

Summary
Displays the parameter dialog of an AddIn and returns the parameters
Syntax
Visual Basic
Public Function ExecuteAddInDialog( _
   ByVal AddInID As String, _
   ByVal ParameterInputString As String _
) As IAddInParameter
Parameters
NameValue typeDescription
AddInIDStringID of the AddIn as defined in the *.vai file of the AddIn
ParameterInputStringStringOriginal parameters to be pre-set in the dialog. This is will often be the value of the 'Data' property of an IAddInParameter Object
Return Type
The IAddInParameter Object representing the parameters entered by the user. This is the same object which is accessed within the AddIn code through the global variable 'Parameter'. The AddIn should set the ok Property of this object to 'True' when the user leaves the dialog through 'OK', and to 'False' if he cancels the dialog.

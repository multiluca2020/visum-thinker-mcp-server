IOperation
Summary
With this object the properties of a certain operation can be read and modified.
Object Model
Description

Using the standard method AttValue the general properties of the operation as type of operation (assignment, mode choice,…, see below), instance (incremental assignment, equilibrium assignment,…) etc. can be accessed and changed. For each group of operation parameters there is one access method returning the corresponding operation parameter object.

The Attribute OperationType provovides information about the type of operation according to enumeration.

For the procedure "Open matrix" an OperationType is provided but no parameter object. Use the procedure "Run script" instead, provided as OperationTypeExecuteScript with the IExecuteScriptPara. Refer to the methode IMatrix.Open() for an example.

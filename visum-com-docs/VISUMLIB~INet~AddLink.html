AddLink Method
No for the new link. Use -1 to let Visum assign the next valid No to the new link

Summary
Add a link
Syntax
Visual Basic
Public Function AddLink( _
   ByVal No As Integer, _
   ByVal FromNode As Variant, _
   ByVal ToNode As Variant, _
   Optional ByVal LinkType As Variant _
) As ILink
Parameters
NameValue typeDescription
NoIntegerNo for the new link. Use -1 to let Visum assign the next valid No to the new link
FromNodeVariantFrom-node (INode Object or node No) of the link
ToNodeVariantTo-node  (INode Object or node No) of the link
LinkTypeVariantLink type (ILinkType Object or type no) for the link
Description
Note that altough the method only returns one ILink Object, the reverse link is created simultaneously.
Examples
-
  • Python 
#retreive some context objects from the network
linkType = Visum.Net.LinkTypes.ItemByKey(50)
fromNode = Visum.Net.Nodes.ItemByKey(10)

cnt1 = Visum.Net.Links.Count
newLink = Visum.Net.AddLink(-1,fromNode,20,linkType)	#add the link using both alternative ways of passing nodes (object / No)
cnt2 = Visum.Net.Links.Count

print "number of Links before %d and after creating the new one  %d" % (cnt1,cnt2)

reverseLink = Visum.Net.Links.ItemByKey(20,fromNode)   #retreive the reverse link which was created automatically
print "New link is %d (%d -> %d)" % (int(newLink.AttValue("No")),int(newLink.AttValue("FromNodeNo")),int(newLink.AttValue("ToNodeNo")))
print "Reverse link is %d (%d -> %d)" % (int(reverseLink.AttValue("No")),int(reverseLink.AttValue("FromNodeNo")),int(reverseLink.AttValue("ToNodeNo")))

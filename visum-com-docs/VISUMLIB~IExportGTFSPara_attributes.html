Names in other languages:

Short name Long name Description
DEU VerwendeGTFSSchlüssel Verwende GTFS-Schlüssel Ist diese Option aktiviert, werden beim Export GTFS-Pflichtfelder unter Verwendung der zuvor importierten GTFS-Schlüssel gefüllt. Sind die GTFS-Schlüssel durch Modifikationen nicht mehr eindeutig, erfolgt eine Konfliktbehandlung. Optionale GTFS-Felder werden ebenfalls aus passend benannten benutzerdefinierten Attributen befüllt, falls ein solches existiert.
ENG UseGTFSKeys Use GTFS keys If this option is selected, mandatory GTFS fields will be filled during export using the previously imported GTFS keys. If the GTFS keys are no longer unique due to modifications, conflict handling will be applied. Optional GTFS fields will also be filled from appropriately named user-defined attributes, if such an attribute exists.
FRA

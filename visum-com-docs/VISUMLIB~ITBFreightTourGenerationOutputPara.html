ITBFreightTourGenerationOutputPara
Summary
Parameters controlling the output of the tour generation in the tour-based freight demand model
Object Model
Description
For each demand stratum, a selection of matrices representing total trips, start trips, return trips and distribution trips can be generated. Each required output matrix is represented by an output item attached to this object. Use the AddItem property to append a new output item and then modify its attributes to specify the type and properties of the output item.

Names in other languages:

Short name Long name Description
DEU KoordGrpNr Koordinierungsgruppen-Nummer Nummer der Koordinierungsgruppe. Signalsteuerungen mit identischer KoordGruppenNr. > 0 werden zu einer Koordinierungsgruppe zusammengefasst, solche mit KoordGruppenNr. = 0 werden nie koordiniert. Standardmäßig ist eine Signalsteuerung unkoordiniert.
ENG CoordGrpNo Coordination group number Number of the coordination group. Signal controllers with the same coordination group no. > 0 are grouped together for coordination. Signal controllers with coordination group no. = 0 are not coordinated at all. By default, a signal controller is uncoordinated.
FRA NumGpCoord NuméroGroupeCoord Numéro du groupe de coordination. Les régulations par feux dont NumGpCoord>0 est identique sont regroupées en un groupe de coordination, celles dont NumGpCoord=0 ne sont jamais coordoonnées. Par défaut, une régulation par feux n’est pas coordonnée.

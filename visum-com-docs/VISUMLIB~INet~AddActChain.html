   ByVal Code As String, _

   ByVal ActChainSequence As String _

Parameters
NameValue typeDescription
CodeStringCode of the activity chain
DemandModelVariantDemand model in which to create the AcitvityChain
ActChainSequenceStringActivity chain sequence as a comma-separated list of activity codes
Description

Creates an ActivityChain. Only applies to VISEM demand models.

The activity pairs occuring in the activity chain sequence must already exist in the demand model

Examples
-
  • Python 
#create demandModel of type 'VISEM'
visemModel = Visum.Net.AddDemandModel("DEMO",3)

#create the activities
act1 = Visum.Net.AddActivity("Home",visemModel)
act1.SetAttValue("IsHomeActivity",True)
act2 = Visum.Net.AddActivity("Work",visemModel)
act3 = Visum.Net.AddActivity("Shop",visemModel)

#create activity pairs - these must exist BEFORE activity chains can be created
ap1 = Visum.Net.AddActPair("Home-Work",visemModel,act1,act2)
ap2 = Visum.Net.AddActPair("Work-Home",visemModel,act2,act1)
ap3 = Visum.Net.AddActPair("Work-Shop",visemModel,act2,"Shop")
ap4 = Visum.Net.AddActPair("Shop-Home","DEMO","Shop","Home")

#create some activity chains
actChain1 = Visum.Net.AddActChain("H-W-H",visemModel,"Home,Work,Home")
actChain2 = Visum.Net.AddActChain("H-W-S-H",visemModel,"Home,Work,Shop,Home")

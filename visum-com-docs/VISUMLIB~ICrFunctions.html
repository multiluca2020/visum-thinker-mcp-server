ICrFunctions
Summary
Provides access to the general settings concerning CR functions.
Object Model
Description

 

Provides access to the CR functions assigned to links and turns.

The method AttValue can be used similar to net objects.

Use attribute 'VdFunctionNo_LinkType(<link type number>)' in order to associate a link type with a CR function.

CR functions can currently NOT be created through COM. Use par/xml files to define required set of CR functions instead.

Examples
-
  • VBScript
  • Python 
Set aCRFunction = Visum.Procedures.Functions.CrFunctions.CrFunction(1)
Visum.WriteToLogFile("Function type: " & aCRFunction.AttValue("vdFunctionType"))
Visum.WriteToLogFile("BPR_b: " &  aCRFunction.AttValue("bpr_b"))

aCRFunction.AttValue("vdFunctionType") = "BPR2"
aCRFunction.AttValue("BPR2_a") = 2.1

Visum.WriteToLogFile("CR function for link type 21:" & Visum.Procedures.Functions.CrFunctions.AttValue("VdFunctionNo_LinkType(21)"))
Visum.Procedures.Functions.CrFunctions.AttValue("VdFunctionNo_LinkType(21)") = 1

aCRFunction = Visum.Procedures.Functions.CrFunctions.CrFunction(1)
Visum.WriteToLogFile("Function type: " & aCRFunction.AttValue("vdFunctionType"))
Visum.WriteToLogFile("BPR_b: " &  aCRFunction.AttValue("bpr_b"))

aCRFunction.SetAttValue("vdFunctionType", "BPR2")
aCRFunction.SetAttValue("BPR2_a",2.1)

Visum.WriteToLogFile("CR function for link type 21:" & Visum.Procedures.Functions.CrFunctions.AttValue("VdFunctionNo_LinkType(21)"))
Visum.Procedures.Functions.CrFunctions.SetAttValue("VdFunctionNo_LinkType(21)",1)

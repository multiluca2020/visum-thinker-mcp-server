IOperations
Summary
Access to the procedures of Visum. Corresponds to the operations sequence in the operations sequence view.
Object Model
Description

To access a single procedure step the ItemByKey method is used. Each operation is identified by its position. For example, ItemByKey(3) refers to the third operation defined within the operations order.

For most objects (marked in the figure above) the property „AttValue“ is used to read and modify the parameters similar to the attributes of net objects. The names of the attributes can be found in the ATTRIBUTE.XLS file.
Notice: The “AttValue” set property has a different notation in Python, C# and Java. Please see Appendix A.

ImportOpenStreetMap Method
List of filenames of Openstreetmap files (*.osm, *.osm.bz2) to be imported

Array containing names of parameter files of the import configuration in the following order:

1) *.xml file describing the import worlflow

2) *.cfg file providing import parameters

3) *.net file providing basic settings and defaults, e.g. link types

4) name of a temporary *.net file to be used during the import - file can be deleted afterwards

5) optional *.gpa file providing graphic parameters to be applied after the import

Set to True to prevent deleting the temporary *.net file

Toggle clipping a rectangle from the input files

0 - no clipping
1 - clipping inside (only elements completely inside the clipping box)
2 - clipping extended (elements, that are completely or partially inside the clipping box

Left bound of clipping area
Lower bound of clipping area

Summary
Imports network data from one or more OSM files.
Syntax
Visual Basic
Public Sub ImportOpenStreetMap( _
   ByVal OpenStreetMapFileNames As Variant, _
   ByVal ParamFileNames As Variant, _
   Optional ByVal SaveNetFile As Boolean = False, _
   Optional ByVal Clipping As Integer = 0, _
   Optional ByVal XCoordMin As Double = 0, _
   Optional ByVal YCoordMin As Double = 0, _
   Optional ByVal XCoordMax As Double = 0, _
   Optional ByVal YCoordMax As Double = 0 _

Examples

  • Python 
Visum = win32com.client.Dispatch("Visum.Visum.2021")

#Create parameter OpenStreetMapFileNames
osmFileNames = [os.path.join(os.path.dirname(sys.argv[0]),"KA_Center.osm")]

#Create parameter ParamFileNames
cfgPath = os.path.join(os.environ["ProgramFiles"],"PTV Vision","PTV Visum 2021","Exe","Importer","OSM","Detailed urban network")
paramFileNames = [
os.path.join(cfgPath,"Detailed urban network.xml"),
os.path.join(cfgPath,"Detailed urban network.cfg"),
os.path.join(cfgPath,"Detailed urban network.net"),
os.path.join(os.path.dirname(sys.argv[0]),"KA_Center.net"), # temporary .NET file
"" # optional GPA file
]

saveNetFile = True
clipping = 0

Visum.IO.ImportOpenStreetMap(osmFileNames, paramFileNames, saveNetFile, clipping)

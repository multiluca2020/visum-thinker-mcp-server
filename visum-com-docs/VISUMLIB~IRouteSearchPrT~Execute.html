Execute Method
Network element container of type INetElements with nodes or zones used for route search.
Link attribute ID that will be assigned the index of the link within the path.
Summary
Executes the shortest path search for private transport. If a short path has already been found in a previous search, it will be continued. To perform a new search, first use Clear.

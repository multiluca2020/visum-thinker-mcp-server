TrafficType

TrafficType as used by IRouteTrafficTypeSet Object

Numerical value internal code Summary
1 Internal_Traffic Internal traffic
2 Orig_Traffic Origin traffic
4 Dest_Traffic Destination traffic
8 Through_Trips Through trips
16 External_Trips External trips
32 BypassInternal_Trips Bypass internal trips
64 INTERNAL_AL_NONE Internal trips without analysis lines
128 INTERNAL_AL_INSIDE Internal trips with analysis lines
256 ORIGDEST_AL_NONE Origin and destination trips with no analyis lines
512 ORIGDEST_AL_INSIDE Origin and destination trips with analyis lines exclusively on the network object selection
1024 ORIGDEST_AL_OUTSIDE Origin and destination trips with analyis lines exclusively outside the network object selection
208 ORIGDEST_AL_BOTH Origin and destination trips with analyis lines exclusively on and outside the network object selection
4096 THROUGHTRANSFER_AL_NONE
8192 THROUGHTRANSFER_AL_INSIDE Transferring through trips with analyis lines exclusively on the network object selection
16384 THROUGHTRANSFER_AL_OUTSIDE Transferring through trips with analyis lines exclusively outside the network object selection
32768 THROUGHTRANSFER_AL_BOTH Transferring through trips with analyis lines exclusively on and outside the network object selection
65536 OTHER_TRIPS Other traffic types

Values can be combined into a single Integer

IJunctionEditor
Summary
The IJunctionEditor object represents the node / junction editor.
Description
The IJunctionEditor object represents the node / junction editor. Creating the object opens the junction editor. Then a layout can be loaded or the window size can be changed. To show a node just choose a synchro mode and mark an INone object using the IMarking interface.
Public Methods
 ExitExits the junction editor
 ExportGeometryViewBoxAsSVGExport Box as SVG File
 ExportGeometryViewSVGExport SVG File
 OpenLayoutOpens a junction editor layout from file.
 SetWindowSizeSets the junction editor’s window size.
Public Properties
 SynchroModeSets synchronization between network editor and junction editor to the selected mode similar to lists

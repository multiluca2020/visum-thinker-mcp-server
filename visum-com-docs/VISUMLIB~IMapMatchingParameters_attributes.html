Names in other languages:

Short name Long name Description
DEU Fangradius Fangradius Damit eine Strecke auf einen Eingabepunkt gematcht wird, darf sie maximal SnapRadius vom Eingabepunkt entfernt sein. SnapRadius sollte also sinnvollerweise so sein groß wie die maximale Ungenauigkeit, mit der die Datenpunkte vorliegen.
ENG SnapRadius Snap radius To match a link and an input point, it may be located at a distance of the maximum MatchRadius from the input point. Therefore, MatchRadius should be as big as the maximum inaccuracy of the exiting data points.
FRA RayonCapture Rayon de capture Pour qu’un tronçon corresponde à un point d’entrée, il doit être éloigné au maximum de RayonCapture du point d’entrée. Il est donc judicieux que le RayonCapture soit aussi grand que l’imprécision maximale avec laquelle les points de données sont disponibles.

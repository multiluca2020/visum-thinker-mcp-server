ILink
Summary
ILink provides access to a single link and all of its properties
Object Model
Examples
-
The next example for the example model KA.ver implements the function "Split according to length" in the context menu of any link.
  • Python 
aLink = Visum.Net.Links.ItemByKey(170200,4800112)
aLink.AttValue("LENGTH")
4.6850000000000005
aRelPos = 2/aLink.AttValue("LENGTH")
aRelPos
0.4268943436499466
aX = aLink.GetXCoordAtRelPos(aRelPos)
aY = aLink.GetYCoordAtRelPos(aRelPos)
aLink.SplitAtPosition(aX,aY)
